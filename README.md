# Justify.AI

An Explainable Multilingual NLP Framework for Causal and Bias-Aware Hate Speech Analysis

## Supported Languages
- English
- Hindi
- Marathi
- Bhojpuri
- Marwari

## Installation

1. Clone repository
2. Install dependencies: `pip install -r requirements.txt`
3. Download models: `python scripts/setup_data.py`
4. Run backend: `python src/api/app.py`
5. Run frontend: `streamlit run frontend/app.py`
