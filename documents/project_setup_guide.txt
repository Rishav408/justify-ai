JUSTIFY-AI - PROJECT SETUP GUIDE (CURRENT)
=========================================

Overview
--------
Justify-AI is a classical NLP + Naive Bayes pipeline that performs multilingual hate analysis with
lexicon matching, causality extraction, bias auditing, and risk scoring. The demo UI consumes the
backend outputs in real time.

Pipeline (Current)
------------------
Input Text -> Language Detection -> Language-Specific Preprocessing ->
Hate Lexicon Matching -> Causality Extraction -> Bias Auditing -> Risk Scoring -> Visual Dashboard

Supported Languages
-------------------
- English
- Hindi
- Marathi
- Bhojpuri
- Marwari

Quick Start
-----------
1) Create and activate a Python environment.
2) Install dependencies:
   - python -m pip install -r requirements.txt
3) Start the backend:
   - python -m uvicorn src.api.app:app --reload
4) Open the demo:
   - http://127.0.0.1:8000 (served by FastAPI)

Key API Endpoint
----------------
POST /api/analyze
Body: { "text": "...", "language": "english" }
Returns:
- language, hate_speech_label, confidence
- lexicon_hits (term, severity, count, spans)
- causality_relations (cause, effect, relation, confidence, sentence)
- bias_metrics (per-dimension + overall)
- risk (score, level, breakdown)

Lexicon Files
-------------
- src/lexicon/lexicons/english.json
- src/lexicon/lexicons/hindi.json
- src/lexicon/lexicons/marathi.json
- src/lexicon/lexicons/bhojpuri.json
- src/lexicon/lexicons/marwari.json

Each entry:
{ "term": "...", "severity": "low|medium|high|critical" }

Tests
-----
Install pytest:
- python -m pip install pytest
Run tests:
- python -m pytest

Notes
-----
- NLTK resources may attempt to download on first run. Offline environments log warnings but remain operational.
- Causality and bias are rule-based baselines that can be expanded with more patterns.
- Risk score is weighted (lexical 0.45, causal 0.30, bias 0.25).
