JUSTIFY-AI - QUICK REFERENCE (CURRENT)
=====================================

Goal
----
Explainable, multilingual hate analysis with classical NLP and interpretable signals.

Core Pipeline
-------------
Input -> Language Detection -> Preprocessing -> Lexicon Matching ->
Causality Extraction -> Bias Auditing -> Risk Scoring -> Dashboard

Key Modules
-----------
- Language Detection: src/api/routes.py
- Preprocessing: src/preprocessing/*.py
- Lexicon Matching: src/lexicon/lexicon_matcher.py
- Causality: src/causality/causality_extractor.py
- Bias Auditing: src/bias/bias_auditor.py
- Risk Scoring: src/scoring/risk_scorer.py
- API: src/api/app.py, src/api/routes.py
- UI: frontend/demo.html + frontend/js/demo.js

API Output Fields (POST /api/analyze)
------------------------------------
- text, language
- hate_speech_label, confidence, confidence_calibrated, confidence_band
- metadata (severity, target_group, tone, etc.)
- lexicon_hits
- causality_relations
- bias_metrics
- risk

Testing
-------
- python -m pytest

Languages
---------
English, Hindi, Marathi, Bhojpuri, Marwari
