Developer Operations Manual

Run, validate, and ship the Justify.AI backend with confidence.

This documentation is the technical operating guide for the Justify.AI backend. It covers API behavior, dataset rules, training flow, model artifacts, testing, and troubleshooting.

API Ready 5 Languages Naive Bayes Explainable Outputs

Quick Start

Minimal commands to run backend + demo locally. Designed for developers who need the system running fast.

Install
python -m pip install -r requirements.txt
Backend
python -m uvicorn src.api.app:app --reload
Demo UI
Open frontend/demo.html

Core API

The backend exposes a single analysis endpoint. Requests accept raw text and an optional language hint.

Endpoint

POST /api/analyze

Base URL: http://127.0.0.1:8000

Language Values

english, hindi, marathi, bhojpuri, marwari

Use auto-detect by omitting language.

Payload Fields

text (required), language (optional)

Request
{ "text": "Sample text to analyze", "language": "english" }
Response
{ "language": "english", "hate_speech_label": "non_hate", "confidence": 0.99, "lexicon_hits": ["..."], "causality_relations": ["..."], "bias_metrics": {"overall": 12, "religion": 0}, "risk": {"score": 8, "level": "low"} }
Why the response matters: Use `risk.score` + `bias_metrics` for dashboards, and `lexicon_hits` + `causality_relations` for explainability views.

Explainability Outputs

Every prediction is bundled with interpretable artifacts so analysts can justify the label.

Lexicon Hits

Term, severity, count, and span positions for language-specific lexicons.

Causality

Cause ? effect pairs with relation confidence and sentence context.

Bias Metrics

Demographic co-occurrence scoring with per-dimension details.

Risk Score

Composite 0–100 score with breakdown (lexical, causal, bias).

Dataset Workflow

All datasets live under src/datasets and follow a 17-field schema. Validation is enforced before any training run.

Schema

Fields: text, label, severity, is_implicit, is_sarcasm, target_group, target_type, directness, call_to_action, tone, emotion, profanity_count, platform, domain, region, confidence, annotator_id.

Validate
python scripts/validate_dataset.py
Validation enforces allowed labels, tone values, boolean fields, and confidence bounds (0.70–1.00).

Model Training & Artifacts

Each language has its own Naive Bayes model and vectorizer saved in src/models/saved_models.

Train All
python src/models/naive_bayes_model.py
Artifacts

{lang}_nb_model.pkl

{lang}_vectorizer.pkl

Split

80/20 stratified split, random_state=42

Troubleshooting

NLTK Download Warnings

Warnings appear offline but do not break the pipeline. Safe to ignore in air-gapped setups.

Model Not Found

Run python src/models/naive_bayes_model.py to regenerate model files.

Dataset Validation Failed

Run validation and ensure schema + tone values comply with src/datasets/schema.md.
